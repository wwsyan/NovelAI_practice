| 说明 | 解决方法 |
| :---: | :---: |
| 画高分辨率图片时，如1600×900，极易出现多人多腿 | 正负面tag拉满，如单人：正面(1girl, solo1.4)，负面(multiple girls:1.4)|

| 如何画多件衣服，如大衣+毛衣+衬衫 | 无 |
